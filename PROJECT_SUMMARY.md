## 🚀 Cómo Usar

### Desarrollo Local
```bash
# Instalar dependencias
npm install

# Compilar contratos
npm run compile

# Ejecutar tests
npm test

# Desplegar en red local
npm run deploy

# Iniciar entorno completo
./start-dev.sh
```

### Frontend
```bash
cd frontend
python3 -m http.server 8000
# Abrir http://localhost:8000
```
